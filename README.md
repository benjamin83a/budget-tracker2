# Budget Tracker

A beginner Python project built to learn programming, GitHub workflows, and application security scanning.

## Features

- Manual expense entry
- Monthly budget calculation
- Category totals
- Remaining budget calculation
- Over-budget warning
- CSV export
- Terminal spending breakdown chart

## Why I built this

I built this project to better understand developer workflows, GitHub repositories, and how security tools like Aikido scan software projects.
